Usage: baft restyle [flags] [root-dir]

Regenerate Mermaid styling in existing BAFT.md files.

Flags:
  --color-palette <name>  Styling palette: vibrant, muted, mono, none
  --stdin                 Read one BAFT.md from stdin and write the restyled result to stdout
  --path <file>           File path label for --stdin mode
  --help, -h              Show this help
